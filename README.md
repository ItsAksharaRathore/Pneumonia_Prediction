# 🫁 Pneumonia Prediction System

An AI-powered web application for detecting **Pneumonia from Chest X-ray Images** using **Deep Learning** and **Computer Vision** techniques.
The system analyzes uploaded chest X-ray scans and predicts whether the patient is affected by pneumonia or not with high accuracy.

🔗 **Live Demo:** https://huggingface.co/spaces/AksharaRathore23/pneumonia_prediction

# 📌 Project Overview

Pneumonia is a serious lung infection that can be diagnosed through Chest X-ray analysis.
This project uses a trained **Deep Learning model** to automatically classify chest X-ray images into:

* **Pneumonia**
* **Normal**

The model extracts important visual patterns from X-ray images and generates prediction results instantly.

This AI-powered system can assist healthcare professionals in faster diagnosis and medical decision-making.

---

# ⚙️ Workflow

1. Upload Chest X-ray Image
2. Image Preprocessing
3. Feature Extraction using CNN
4. Deep Learning Model Prediction
5. Display Prediction Result

# 📈 Training Performance

The model was trained for **20 epochs** and achieved excellent validation accuracy during training.

| Epoch | Training Loss | Validation Accuracy |
| ----- | ------------- | ------------------- |
| 1     | 0.1884        | 97.52%              |
| 5     | 0.0484        | 97.71%              |
| 10    | 0.0302        | 95.13%              |
| 15    | 0.0098        | 98.47%              |
| 20    | 0.0026        | **99.14%**          |
