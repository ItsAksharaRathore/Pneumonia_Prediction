# 🫁 Pneumonia Prediction System

An AI-powered web application for detecting **Pneumonia from Chest X-Ray Images** using **Deep Learning** and **Computer Vision** techniques.
This project helps in early diagnosis by analyzing uploaded chest X-ray scans and predicting whether the patient is affected by pneumonia or not.

🔗 **Live Demo:** https://huggingface.co/spaces/AksharaRathore23/pneumonia_prediction

## 📌 Project Overview

Pneumonia is a serious lung infection that can be identified through Chest X-ray imaging.
This project uses a trained **Convolutional Neural Network (CNN)** model to classify X-ray images into:

* **Pneumonia**
* **Normal**

The system processes uploaded images, extracts important visual features, and generates prediction results instantly.
